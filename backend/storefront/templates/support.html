🎧 Developer Support & Odra Integration

Integrate Fortis AI's autonomous security guardrail into your DeFi agent workflow in under 5 minutes using Casper's HTTP 402 protocol and the Odra Rust Framework.

🐍   Python Agent Integration — Quickstart
SDK V1.0 🟢

Before executing any transaction on Casper, your AI agent sends a scan request. If HTTP 402 is returned, automatically pay the CSPR invoice and attach the proof header:

import requests
# 1. Request Compliance Verification from Fortis AI
res = requests.post("http://fortis.casper:8000/api/v1/scan/", json=payload)
if res.status_code == 402:
invoice = res.json()["invoice"]
proof = casper_wallet.pay(invoice["treasury_address"], invoice["amount_cspr"])
# 2. Re-submit with x402 Payment Proof
res = requests.post(..., headers={"X-Casper-402-Proof": proof})
token = res.json()["compliance_token"] # Proceed with trade!
🦀   Odra Framework Smart Contract Verification
RUST KERNEL 🟠

In your target DeFi contract on Casper, invoke the Odra bridge to verify the immutable SHA-256 compliance hash before releasing yield:

#[odra::module]
impl YieldVault {
pub fn execute_trade(&mut self, token_jwt: String, proof_hash: String) {
let is_verified = self.fortis_log.verify_hash(proof_hash);
if !is_verified {
odra::revert(Error::SecurityRiskDetected);
}
// ✅ Release capital safely!
}
}
